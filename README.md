# powerbi-sales-dashboard-dataset
Sales Dashboard using Power BI dataset
# 📂 Dataset Information

## 📊 Overview
This dataset is used for analyzing sales performance across different regions, categories, and customer segments. It serves as the foundation for building the Power BI Sales Dashboard.

## 🗂️ Dataset Details
The dataset contains information related to:
- Sales transactions
- Product categories
- Customer segments
- Shipping details
- Regional performance

## 📌 Key Columns
- **Order ID** – Unique identifier for each order  
- **Order Date** – Date of purchase  
- **Region** – Sales region (Central, East, South, West)  
- **Category** – Product category (Furniture, Office Supplies, Technology)  
- **Sub-Category** – Detailed product classification  
- **Sales** – Revenue generated  
- **Profit** – Profit earned  
- **Quantity** – Number of items sold  
- **Discount** – Discount applied  
- **Ship Mode** – Shipping method used  

## 🧹 Data Preprocessing
- Removed missing/null values  
- Standardized column formats  
- Converted date fields into proper format  
- Ensured numerical consistency for calculations  

## 📈 Purpose of Dataset
The dataset is used to:
- Analyze sales trends over time  
- Compare performance across regions  
- Identify high-performing product categories  
- Support data-driven decision-making  

## ⚠️ Note
This dataset is used for educational and project purposes only.
